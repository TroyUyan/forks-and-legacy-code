GP Winner Badges

This icon set was created for use at the Precision Racing League website, an online racing league. The initial intention was for drivers to display them as achievement badges in their forum signature. However the concept was quite attractive so I branched the work to other websites and leagues, creating a large collection of icons.

They are yours to use for anything you would like! All I ask is that please host them on your own server instead of hotlinking to mine. :) If you like the badges and would like to say thank you, request a special badge, new track set, update, or a name tag, please feel free to get in touch with me via the contact page.
